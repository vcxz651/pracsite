# 튜토리얼 페이지 기획안

> 상태: 보류/아카이브
> 현재 우선 문서: `docs/demo_page_plan.md`

이 문서는 과거의 `/tutorial/` 전용 독립 샌드박스 구상을 기록한 보류 문서다.
현재 프로젝트의 사용자 안내 우선순위는 데모 페이지 재구성 쪽에 있으며, 이 문서를 구현 지침의 단일 기준으로 사용하지 않는다.

## 현재 판단

- `/tutorial/` 전용 독립 플로우는 현재 구현 우선순위가 아니다.
- 신규 사용자 안내는 데모 페이지의 시나리오 선택 + 투어 안내 방식이 우선이다.
- 이 문서에 있던 상세 URL, 단계별 샌드박스, 템플릿/배너 분해 계획은 모두 **과거 아이디어**로만 본다.

## 유지 이유

- 향후 별도 튜토리얼을 재검토할 때 참고할 수 있는 기획 축이었기 때문이다.
- 다만 현재 구현과 혼동되지 않도록, 세부 실행 항목은 본 문서에서 제거했다.

## 재개 조건

다음 조건이 모두 충족되면 별도 튜토리얼 기획을 다시 연다.

1. 데모 페이지 A/B/C 구조가 먼저 안정화될 것
2. 데모 투어만으로는 신규 유저 온보딩이 부족하다는 근거가 생길 것
3. `/tutorial/`를 실제로 라우팅/운영할 명확한 범위가 정리될 것

그 전까지는 `docs/demo_page_plan.md`를 기준으로 판단한다.
